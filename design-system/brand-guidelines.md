# TrueNorth Brand Guidelines

## Brand Positioning

**North Star**: A founder can move from rough idea to a credible first launch with a public surface, operating plan, and outreach-ready assets.

**Value Proposition**: We transform early-stage concepts into launch-ready foundations that inspire confidence and drive action.

## Tone & Voice

### Core Attributes
- **Clear**: We communicate with precision and avoid unnecessary jargon
- **Confident**: We project capability without arrogance
- **Supportive**: We empower founders to take action
- **Professional**: We maintain credibility while staying approachable

### Voice Principles
- Use active voice and direct language
- Prioritize clarity over cleverness
- Guide rather than dictate
- Celebrate progress over perfection

## Visual Identity

### Logo Usage
- Primary logo: Navy (#1a2332) on white or cream backgrounds
- Reverse logo: White or gold (#d4a843) on dark backgrounds
- Minimum clear space: Equal to the height of the "T" in TrueNorth
- Minimum size: 24px height for digital, 0.5" for print

### Color Palette

#### Primary Colors
- Navy (#1a2332): Primary brand color, headers, footers
- Gold (#d4a843): CTAs, accents, highlights
- Cream (#faf8f0): Backgrounds, light sections

#### Secondary Colors
- Charcoal (#2d3748): Body text, secondary elements
- Sage (#687864): Success states, positive messaging
- Coral (#e07a5f): Error states, urgent alerts

#### Functional Colors
- Blue 600 (#0284c7): Links, interactive elements
- Green 500 (#10b981): Success confirmations
- Red 500 (#ef4444): Error messages

### Typography

#### Headings: Playfair Display
- Use for all headings (H1-H6)
- Weights: 700 (bold), 800 (extrabold)
- Letter spacing: -0.02em for H1-H2

#### Body Text: Inter
- Use for all body copy, UI elements, forms
- Weights: 400 (regular), 500 (medium), 600 (semibold), 700 (bold)
- Line height: 1.5 for body, 1.25 for headings

#### Code/Technical: JetBrains Mono
- Use for code snippets, technical specifications
- Weights: 400 (regular), 500 (medium)

### Imagery Guidelines
- Use authentic photography when possible
- Prefer natural lighting and real environments
- Avoid generic stock photography
- Maintain consistent filter/warmth across all images

## Application Guidelines

### Digital Applications
- Maintain 8px grid system for all layouts
- Use 16px minimum for body text
- Ensure 4.5:1 minimum contrast ratio (WCAG AA)
- Implement responsive breakpoints consistently

### Print Applications
- Convert hex colors to CMYK/Pantone as needed
- Use 300 DPI minimum for all raster images
- Include 0.125" bleed for print materials

## Do's and Don'ts

### Do
- Use the official color palette
- Maintain proper clear space around logo
- Use approved typography combinations
- Ensure accessibility compliance

### Don't
- Stretch or distort the logo
- Use colors outside the approved palette
- Place logo on busy backgrounds
- Use more than two typefaces in one layout

## Brand Assets

All brand assets are available in the `/design-system/assets` directory:
- Logo files (SVG, PNG, EPS)
- Color palette (ASE, SKP)
- Typography files (when self-hosting)
- Templates (PowerPoint, Word, Figma)
